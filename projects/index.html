Projects

우리 연구실(DAISY LAB)은 의료 인공지능의 설명 가능성을 높이고, 다중 모달리티 데이터를 융합하여 실제 임상 현장에서 신뢰할 수 있는 차세대 진단 및 예측 모델을 개척하고 있습니다.

Research Interests

설명가능 진단 모델 개발 Explainable Diagnostic Model

의료 영상, 생체신호, 임상 기록 등 다양한 데이터를 기반으로 진단 결과의 근거를 해석 가능하게 제시하는 설명가능 인공지능 모델을 개발합니다.

xai medical-imaging diagnostics
설명가능 예후 예측 모델 개발 Explainable Prognosis Prediction Model

환자의 임상 경과를 예측함과 동시에 예측 결과의 주요 요인을 정량적으로 설명할 수 있는 설명가능 예후 예측 모델을 개발합니다.

xai prognosis predictive-modeling
멀티모달 의료 리포트 생성 모델 개발 Multimodal Medical Report Generation Model

의료 영상, 임상 기록을 통합한 멀티모달 데이터를 활용하여 임상적으로 유의미한 의료 리포트를 자동으로 생성하는 모델을 개발합니다.

multimodal report-generation medical-imaging
다중 사이트 연합학습 Multi-site Federated Learning

여러 의료 기관에 분산된 데이터를 직접 공유하지 않고도 학습이 가능하도록 개인정보 보호와 데이터 보안을 고려한 다중 사이트 연합학습 기법을 연구합니다.

federated-learning privacy-preserving data-security

결측 데이터 보완과 비용 효율성 향상을 위해 서로 다른 의료 영상 모달리티 간 변환을 수행하는 인공지능 모델을 개발합니다.

modality-translation generative-ai medical-imaging
